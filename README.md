# ionic2-devise-token-auth

## Installation

To install this library, run:

```bash
$ npm install ionic2-devise-token-auth --save
```

## Development

To generate all `*.js`, `*.js.map` and `*.d.ts` files:

```bash
$ npm run tsc
```

To lint all `*.ts` files:

```bash
$ npm run lint
```
